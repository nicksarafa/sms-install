### SMS Install **In Progress**

App landing page for easy sms install link dispatching for iOS and Android apps inspired by www.hopper.com 

#### Dependencies

* Gulp
* Node
* Twilio

#### Gulp Tasks

##### Convert HAML to HTML

`gulp haml`

##### Convert SASS to CSS 

`gulp sass`

##### Run all gulp tasks in parralel while developing 

`gulp watch`
